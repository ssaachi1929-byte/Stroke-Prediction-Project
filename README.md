# Stroke-Prediction-Project
# This project focuses on identifying the factors that contribute to stroke risk among adults aged 50 and above. The objective is to predict the likelihood of stroke using demographic, lifestyle, and medical features, including gender, age, hypertension, heart disease, martial status, work type, residence type, average gluocose level, BMI, and smoking status. The dataset included older adults with and without a history of stroke, enabling the model to identify patterns associated with stroke risk and support early detection and preventive healthcare interventions.
